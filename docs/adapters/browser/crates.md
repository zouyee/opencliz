# crates.io

**Mode**: 🌐 Public · **Domain**: `crates.io`

## Commands

| Command | Description |
|---------|-------------|
| `opencliz crates search` | Search crates by query |
| `opencliz crates info` | Crate metadata by name |

## Examples

```bash
opencliz crates search --query serde -f json
opencliz crates info --crate serde -f json
```
