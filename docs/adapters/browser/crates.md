# crates.io

**Mode**: 🌐 Public · **Domain**: `crates.io`

## Commands

| Command | Description |
|---------|-------------|
| `opencli crates search` | Search crates by query |
| `opencli crates info` | Crate metadata by name |

## Examples

```bash
opencli crates search --query serde -f json
opencli crates info --crate serde -f json
```
