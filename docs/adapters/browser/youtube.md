## Usage Examples

```bash
# Quick start
opencliz youtube search --limit 5

# JSON output
opencliz youtube search -f json

# Verbose mode
opencliz youtube search -v
```

## Prerequisites

- Chrome running and **logged into** youtube.com
- [Browser Bridge extension](/guide/browser-bridge) installed
