## Usage Examples

```bash
# Quick start
opencli zhihu hot --limit 5

# JSON output
opencli zhihu hot -f json

# Verbose mode
opencli zhihu hot -v
```

## Prerequisites

- Chrome running and **logged into** zhihu.com
- [Browser Bridge extension](/guide/browser-bridge) installed
