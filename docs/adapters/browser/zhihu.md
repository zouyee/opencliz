# Zhihu

**Mode**: 🔐 Browser · **Domain**: `zhihu.com`

## Commands

| Command | Description |
|---------|-------------|
| `opencliz zhihu hot` | |
| `opencliz zhihu search` | |
| `opencliz zhihu question` | |
| `opencliz zhihu download` | |

## Usage Examples

```bash
# Quick start
opencliz zhihu hot --limit 5

# JSON output
opencliz zhihu hot -f json

# Verbose mode
opencliz zhihu hot -v
```

## Prerequisites

- Chrome running and **logged into** zhihu.com
- [Browser Bridge extension](/guide/browser-bridge) installed
