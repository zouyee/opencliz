# Steam

**Mode**: 🌐 Public · **Domain**: `store.steampowered.com`

## Commands

| Command | Description |
|---------|-------------|
| `opencliz steam top-sellers` | Top selling games on Steam |

## Usage Examples

```bash
# Quick start
opencliz steam top-sellers

# Limit results
opencliz steam top-sellers --limit 5

# JSON output
opencliz steam top-sellers -f json
```

## Prerequisites

- No login required (public API)
