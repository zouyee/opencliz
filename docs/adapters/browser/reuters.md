# Reuters

**Mode**: 🔐 Browser · **Domain**: `reuters.com`

## Commands

| Command | Description |
|---------|-------------|
| `opencliz reuters search` | |

## Usage Examples

```bash
# Quick start
opencliz reuters search --limit 5

# JSON output
opencliz reuters search -f json

# Verbose mode
opencliz reuters search -v
```

## Prerequisites

- Chrome running and **logged into** reuters.com
- [Browser Bridge extension](/guide/browser-bridge) installed
