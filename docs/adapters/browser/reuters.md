|---------|-------------|

## Usage Examples

```bash
# Quick start
opencli reuters search --limit 5

# JSON output
opencli reuters search -f json

# Verbose mode
opencli reuters search -v
```

## Prerequisites

- Chrome running and **logged into** reuters.com
- [Browser Bridge extension](/guide/browser-bridge) installed
