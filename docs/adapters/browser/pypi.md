# PyPI

**Mode**: 🌐 Public · **Domain**: `pypi.org`

## Commands

| Command | Description |
|---------|-------------|
| `opencli pypi search` | Search packages (JSON API) |
| `opencli pypi info` | Package metadata JSON |

## Examples

```bash
opencli pypi search --query requests -f json
opencli pypi info --package requests -f json
```
