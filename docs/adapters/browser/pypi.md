# PyPI

**Mode**: 🌐 Public · **Domain**: `pypi.org`

## Commands

| Command | Description |
|---------|-------------|
| `opencliz pypi search` | Search packages (JSON API) |
| `opencliz pypi info` | Package metadata JSON |

## Examples

```bash
opencliz pypi search --query requests -f json
opencliz pypi info --package requests -f json
```
