# SMZDM

**Mode**: 🔐 Browser · **Domain**: `smzdm.com`

## Commands

| Command | Description |
|---------|-------------|
| `opencliz smzdm search` | |

## Usage Examples

```bash
# Quick start
opencliz smzdm search --limit 5

# JSON output
opencliz smzdm search -f json

# Verbose mode
opencliz smzdm search -v
```

## Prerequisites

- Chrome running and **logged into** smzdm.com
- [Browser Bridge extension](/guide/browser-bridge) installed
